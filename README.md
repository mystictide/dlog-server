DLog Server is a Web API that serves as the backend for the web app [DLog](https://github.com/mystictide/dlog-client).

### Technologies Used
DLog is built or managed using the following technologies:

.NET Core  
Dapper  
Docker  
PostgreSQL   
Self-hosted on Ubuntu Server
